GitHub Pages & Supabase

爆速で認証・ランキング機能付きWebアプリを公開しよう！

サーバーの知識がなくても、無料で高機能なWebアプリを公開できる。そんな夢のような開発スタイルをインフォグラフィックで解説します。

こんな悩み、ありませんか？

「自分のサイトにログイン機能やランキングを実装したい…でも、サーバーの準備や管理は難しそうだし、お金もかかりそう…」

🤔

その悩み、この2つが解決します！

GitHub Pages

🚀

静的サイトの王様

リポジトリから直接、無料でWebサイトをホスティング。

Supabase

⚡️

バックエンドの救世主

認証・DB・ストレージをコードを書かずに構築。

なぜこの組み合わせが最強なのか？

静的なサイトしか公開できないGitHub Pagesの弱点を、SupabaseのAPIが見事に補完。フロントエンド開発に集中できる、理想的なワークフローが完成します。

🖥️

1. フロントエンド

HTML/CSS/JSで見た目を作成し、GitHub Pagesで公開。

🔗

2. 連携

⚙️

3. バックエンド

ユーザー情報やスコアの管理はSupabaseに丸投げ。
